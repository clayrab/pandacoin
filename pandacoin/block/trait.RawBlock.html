Expand description

This structure is a basic block, it should be 1-to-1 with the physical data which will be serialized and send over the wire and stored on disk. We provide a useless default implementation for the sake of making different mock RawBlocks easier. We would prefer to define only the interface here and the default implementation in mock_block.rs, but Rust does not allow this.
